Git学习笔记

git init //初始化一个git仓库
git add
git commit
git status
git diff

git reset –hard commit_id //选择版本
git log //查看提交历史
git reflog //查看命令历史

git checkout – file //丢弃工作区的修改
git reset HEAD file //回到工作区

git rm //删除文件

关联远程库和第一次推送，可用GitHub上的命令——分两种情况：
1、从远程clone到本地:
git clone git@github.com:userName/repositoryName.git
2、本地有一个git仓库，远程(Github)创建一个，然后切换到本地目录通过以下命令与远程关联：
git remote add origin git@github.com:userName/repositoryName.git
“origin”是给远程仓库取得名字

git push origin master //推送修改
git pull origin master //将远程的代码更新到本地

git branch //查看本地分支列表
git branch -r //查看远程分支列表
git branch Name //创建分支
git checkout Name //切换分支
git checkout -b Name //创建+切换分支
git merge Name //合并分支到当前分支
git branch -d Name //删除分支
git branch -D Name //强制删除分支

git tag //打标签

git log –graph //查看分支合并图

git lg == git config –global alias.lg “log –color –graph –pretty=format:’%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset’ –abbrev-commit” //配置log别名

find . -name “.git” | xargs rm -Rf //删除git仓库中的隐藏”.git”文件

git stash //暂存
git stash list
git stash apply
git stash drop
git stash pop
git stash clear
