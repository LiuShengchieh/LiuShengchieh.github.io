Mac软件推荐

用了近一年的 Macbook Pro，已经离不开它了。真是生活工作学习必备之良品啊。

如果你将要买苹果电脑或者刚买，那么不妨看看此文。推荐一些个人觉得好用的软件，而Mac本身的使用技巧————触控板、快捷键等，苹果官网有，链接置于文章底部。

软件：

Sublime Text
可切换多种文本格式的编辑器，写代码、写文章、记笔记，均可。

Evernote
印象笔记，很有名的一个笔记软件。免费账户可以同步两个设备，每月有60MB的上传流量，光是文字的话足够了吧？千万不要用「有道云笔记」，Mac上有三大缺点：
1、打字卡顿
2、光标不定期乱跳
3、标题不定期同步失败
很影响使用的！

Xmind
画思维导图的软件，学着用用吧，整理思路、列事项，很有效。

CheatSheet
长按”command”可查看处于当前窗口的软件的快捷键。

MPlayerX
视频播放器，基本上支持所有的视频格式。

网易云音乐
网上好多人推荐，想知道具体好在哪去知乎看看？反正我就是一直在用啦。

App Cleaner
免费版的可以卸载软件，删除大部分垃圾文件。有一小部分垃圾文件需要升级到pro版才会帮你删除，不过它会列出文件位置，不想付费又不怕麻烦的话可以通过Terminal进到对应的文件夹里去手动删除。

The Unarchiver
解压缩软件。

欧路词典
很简洁的小词典，通过快捷键可以快速弹出查询窗口，超级方便实用。

Thunder
迅雷。Mac版的没有广告诶！

其他
微信、QQ、Android Studio等，这个各取所需，就不特意推荐了。
